# Bulk File Organizer

A Python tool that automatically organizes files into folders based on file type.

## Features

- Organizes images, videos, documents, and audio files
- Creates folders automatically
- Handles unknown file types
- Simple command-line interface

## Technologies

- Python
- os module
- shutil module

## How to Run

python organizer.py
